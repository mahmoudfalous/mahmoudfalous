## 🧠 Professional Summary

I am a **Full-Stack Software Engineer** with a strong emphasis on **backend architecture**, **API design**, and **scalable systems**. 

Currently creating production-ready applications using **Ruby on Rails** and **Next.js**, I am passionate about clean code, robust testing with RSpec, and long-term maintainability.

---

### 🎯 What I’m Focused On

* 🔧 **Architecture:** Designing modular **Rails APIs** & Microservices.
* 🧪 **Reliability:** Writing comprehensive tests (RSpec, Integration & Unit).
* ⚡ **Performance:** Background processing (Sidekiq) & Database optimization.
* 🐳 **DevOps:** Containerization with **Docker** & deployment via **Kamal**.
* 💳 **Integrations:** Complex payment flows (Stripe Connect) & Real-time systems (Firebase).
