I am a **Full-Stack Software Engineer** focused on designing and building production-ready systems.

My primary expertise lies in **Ruby on Rails**, **backend architecture**, **API design**, **database optimization**, and **scalable application development**. I enjoy solving complex engineering challenges while keeping systems maintainable, testable, and easy to evolve.

Currently building full-stack applications using **Rails**, **Next.js**, and **Flutter**, while continuously improving system design, DevOps practices, and distributed architectures.
